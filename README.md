# team-based-eeg-classification-braindecode-bci-competition
This repository contains the implementation for classifying motor imagery tasks (right-hand vs. right-foot) using EEG signals from the BCI Competition III Dataset IVa. The project utilizes several deep learning models provided by the Braindecode library, including EEGNet, ShallowFBCSPNet, Deep4Net, and EEGConformer.
